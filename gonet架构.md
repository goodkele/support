## 1. 模块

### 1.1 模块介绍

0. 启动中心hub服务
1. 启动逻辑服务向hub注册
2. 启动聊天服务向hub注册
3. 启动网关服务向hub注册，socket连接到逻辑服务和聊天服务
4. 开始对外提供服务

### 1.2 模块启动流程

* hub 中心服务。监控所有服务状态，负载，在线人数
* gate 网关服务。所有用户socket连接到此服务。
* game 逻辑服务。游戏主逻辑
* chat 聊天服务。
